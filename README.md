Clock App

A simple web application created as a high school project.

 Features
	•	Digital and analog clock display
	•	Toggle between clock modes
	•	Weather display

 Technologies Used
	•	HTML
	•	CSS
	•	JavaScript

 How to Run
	1.	Download or clone the repository
	2.  Add your OpenWeather API key in fetchWeather()
  3.	Open index.html in your browser

 About

This project was built as part of my learning journey in web development during high school.
